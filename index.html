MODembed

Model-based development environment for embedded systems. Currently supports the 8-bit PIC18 microcontroller family.

The project aims to create a full-featured IDE which can support every aspect of an embedded system design using a model-based approach.

Features

The currently implemented capabilities of the tool is the following:

  • A modular assembly language and assembler tool which enables reusable software components - so-called modules to be compiled into a program that can be run by a microcontroller.
  • Easy to use code editor with syntax highlighting and content assist
  • Configuration model with convenient editor for configuration bits and component configuration
