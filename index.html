MODembed

MODembed is an experimental IDE aiming to provide a more advanced toolchain for embedded software development. It provides a model-based approach for defining the whole compilation process.

The project aims to create a full-featured IDE which can support every aspect of an embedded system design using a model-based approach.

Features

Model-based approach

Everything is a model: program, device and architecture

Modeling workflow

User gets full control over compilation process

Architecture-independent toolchain

The target architecture is described by a model, all tools are designed to be fully independent of the target

Reverse engineering

Tools to support disassembling programes are included

Included libraries

Architecture models and libraries are available for Microchip's PIC midrange (PIC16), enchanced midrange (PIC16e) and high-end (PIC18) microcontrollers.
